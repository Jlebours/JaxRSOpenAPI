## How to install
- Install Maven
- Install Java 15 (15.0.2 used)
- Install MySQL 8 

## How to run
Open phpMyAdmin in localhost , create : 
- a database named '**mydatabase**'
- a user with login '**root**' and no password

http://localhost:8082/api/
